# authflow-ui-project
Project Overview

AuthFlow is a modern static authentication user interface project developed as part of an Open Source Software Development group assignment. The project demonstrates collaborative software development using Git and GitHub while providing a clean and responsive authentication interface.

Project Objectives

- Practice collaborative software development using Git and GitHub.
- Understand branching, pull requests, code reviews, and merging.
- Build a modern authentication user interface.
- Improve teamwork and version control skills.
- Apply open-source development workflows in a real project.

Features

Landing Page

- Responsive navigation bar
- Hero section
- Features showcase
- Call-to-action buttons
- Footer section

Login Page

- Email input field
- Password input field
- Remember Me option
- Login button
- Forgot Password link

Signup Page

- Full Name field
- Email field
- Password field
- Confirm Password field
- Create Account button

Forgot Password Page

- Email input field
- Password reset request form
- Back to Login link

Responsive Design

- Mobile-friendly layouts
- Tablet support
- Desktop optimization

Technologies Used

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
